In this directory, we have define different just as an example of what would they look like. If you read the code line by line you will notice in the Aftab class classes are being created on the fly using the base `PQNAgent` class here. 

This choice in architecture happened because we wanted researchers to be able to play with different aspects of code including the pre-defined encoders, thereby we needed to pass the encoders at the instance of creating the agent. 

However, this codes could serve to great educational ends. Alternatively, they could just be copied and pasted and changed as researchers wish.
